📖 MiMo v2 TTS API 文档

📡 接口信息

项目值
模型mimo-v2-audio-tts
API 端点https://api.xiaomimimo.com/v1/chat/completions
认证方式Header api-key: YOUR_KEY
输出格式WAV (24kHz, 16-bit PCM, 单声道)
最大文本长度10,000 字符

📤 请求格式

curl -X POST https://api.xiaomimimo.com/v1/chat/completions \
  -H "Content-Type: application/json" \
  -H "api-key: YOUR_API_KEY" \
  -d '{
    "model": "mimo-v2-audio-tts",
    "audio": {"format": "wav", "voice": "mimo_default"},
    "messages": [
      {"role": "assistant", "content": "你好，我是小米AI语音助手。"}
    ]
  }'

🎨 风格能力一览

分类示例说明
情绪开心 悲伤 生气控制语音情感基调
方言东北话 四川话 粤语模拟不同地区口音
角色像个大将军 像个小孩 孙悟空模仿特定角色声音
组合深情款款 语速慢 撒娇 台湾腔多个风格自由组合
唱歌唱歌 + 歌词内容开启歌唱模式

📥 响应格式

{
  "choices": [{
    "message": {
      "audio": { "data": "<base64-encoded-wav>" }
    }
  }]
}

🐍 Python 示例

import requests, base64

resp = requests.post(
    "https://api.xiaomimimo.com/v1/chat/completions",
    headers={
        "Content-Type": "application/json",
        "api-key": "YOUR_API_KEY"
    },
    json={
        "model": "mimo-v2-audio-tts",
        "audio": {"format": "wav", "voice": "mimo_default"},
        "messages": [
            {"role": "assistant", "content": "今天天气真不错！"}
        ],
    },
)

audio = base64.b64decode(resp.json()["choices"][0]["message"]["audio"]["data"])
with open("output.wav", "wb") as f:
    f.write(audio)
